## 🚀 Interactive Modules

### 1. The ACH Network Deep-Dive
* **The Problem:** Deciphering the archaic 94-character NACHA format.
* **The Interactive Asset:** A real-time NACHA file viewer and batch-processing simulator.
* **Key Learnings:** Settlement windows, ODFI/RDFI roles, and return code logic.

### 2. AIOps: Noise to Insight
* **The Problem:** Alert fatigue in mission-critical banking environments.
* **The Interactive Asset:** An incident simulator that demonstrates alert suppression and automated Root Cause Analysis (RCA).
* **Key Learnings:** ML anomaly detection, event correlation, and auto-remediation policies.

## 🛠️ The Stack
This project is built using a "Tech-Native" aesthetic, designed to be lightweight, responsive, and embeddable within internal corporate wikis.
* **Frontend:** Vanilla HTML5, CSS3 (Custom "Financial Newspaper" Design System).
* **Interactivity:** Vanilla JavaScript (ES6+).
* **Deployment:** GitHub Pages (CI/CD via GitHub Actions).
* **Design Tokens:** Custom Barlow Condensed & Source Code Pro typography for a professional terminal-meets-journal feel.
